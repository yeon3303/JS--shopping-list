## DreamCoding JS Browser Course - Shopping List 실습

### - 필요한 기능

1. Input에 아이템을 입력.
2. +버튼 or Enter로 아이템을 추가.
3. 추가된 아이템은 리스트에 업데이트.
4. 휴지통 버튼으로 리스트 아이템을 삭제.
5. Clear 버튼으로 리스트의 모든 아이템 삭제.
